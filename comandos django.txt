django-admin startproject <nombre>      => crear un proyecto de django
python manage.py                        => sirve para utilizar el CLI de django pero ya en el proyecto en especifico
python manage.py startapp <nombre>      => sirve para crear una aplicacion dentro de mi proyecto de Django
python manage.py runserver              => levanta el proyecto en modo de desarrollo
python manage.py showmigrations         => mostrara todas las migraciones creadas en nuestro proyecto y las que ya se han ejecutado tendran una [X]
